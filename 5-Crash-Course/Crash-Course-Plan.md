## Learning Progression

```
Sessions 1-3    → Bootstrap Layout (Navbar, Hero, Grid)      [No JavaScript yet]
Session 4       → JavaScript Foundations                       [Dedicated JS intro]
Sessions 5-7    → Bootstrap Components + JS Interaction       [Cards, Tables, Gallery + DOM]
Sessions 8-9    → Forms + Validation                          [Bootstrap Forms + JS Regex]
Sessions 10-11  → Data + Dynamic Features                     [localStorage, Timers, Dark Mode]
Session 12      → Code Organization                           [External JS, Refactoring]
Sessions 13-14  → Polish + Testing                            [Custom CSS, Responsive, A11y]
Session 15      → Launch Day                                  [Showcase + Deployment]
```

## JavaScript Progression (ES5 → ES6+)

| Sessions | Style | Features Used |
|----------|-------|---------------|
| 1-3 | No JavaScript | Pure HTML + Bootstrap layout |
| 4-8 | ES5 (classic) | `var`, `function() {}`, `"Hello " + name` |
| 9-12 | ES6+ introduced | `let`, `const`, `` `Hello ${name}` ``, `addEventListener` |
| 13-15 | Full modern | Arrow functions, `forEach`, `DOMContentLoaded` |
