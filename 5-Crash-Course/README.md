## 📖 What Is This Crash Course?

Instead of learning Bootstrap and JavaScript separately, you'll build **one real website** — section by section, session by session. Each session adds a new part to the website while teaching you new skills. By the end, you'll have a **complete, professional-quality, responsive one-page website** showcasing your BCA class!

**What you'll learn:**
- ✅ Bootstrap 5 — responsive layouts, components, forms, and utilities
- ✅ JavaScript — DOM manipulation, events, validation, localStorage, and dynamic effects
- ✅ HTML & CSS fundamentals — reinforced through practical use (every new tag is explained!)
- ✅ Real-world skills — code organization, testing, accessibility, and deployment

## 🚦 How to Use This Crash Course

### For Each Session:

1. **Open the session file** (e.g., `Session-01-Setup-and-Navbar.md`)
2. **Read the "What We'll Build Today"** section — see the goal
3. **Check the HTML & CSS Quick Reference** — new tags you'll learn
4. **Follow the step-by-step lab** — type the code yourself (don't copy-paste!)
5. **Look for 📌 callouts** — these explain HTML/CSS tags the first time they appear
6. **Complete the checkpoint** — make sure your page matches the expected output
7. **Try the practice challenges** — optional stretch exercises

### Tips for Success:

- 🖊️ **Type every line** — muscle memory matters more than reading
- 🔍 **Use browser DevTools** (F12) — inspect elements, check console for errors
- 💾 **Save frequently** (Ctrl+S) and refresh browser (F5) to see changes
- 🤔 **Read the 📌 callouts** — they explain why, not just how
- 👥 **Help classmates** — teaching others reinforces your own learning

## 📖 JavaScript Progression

We use a **progressive approach** to JavaScript — starting simple, getting more powerful:

| Sessions | Style | What You'll Use |
|----------|-------|-----------------|
| 1-8 | ES5 (classic) | `var`, `function() {}`, `"Hello " + name` |
| 9-12 | ES6+ (modern) | `let`, `const`, `` `Hello ${name}` `` (template literals) |
| 13-15 | Full modern | Arrow functions, `forEach`, modern patterns |

This way you can read both older code online AND write modern code!
